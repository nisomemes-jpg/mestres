# Site Mestre de Cerimônia

Site estático pronto para GitHub Pages.

## Como publicar pelo GitHub

1. Crie um repositório no GitHub.
2. Envie todos os arquivos desta pasta para o repositório.
3. No GitHub, abra Settings > Pages.
4. Em Branch, escolha main e /root.
5. Salve. O GitHub vai gerar o link público do site.

O arquivo principal é index.html. A apostila em Word fica em downloads/.
